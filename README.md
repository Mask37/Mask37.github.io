# hexo

基于hexo + next 修改为自己的个人站点

# 使用到的插件

| 命令                                 | 描述              |
| ------------------------------------ | ----------------- |
| npm install hexo-deployer-git --save | 部署至GitHub page |
|                                      |                   |
|                                      |                   |

# 使用步骤
1. clone 本项目到本地
2. npm install
3. 下载主题
```bash
git clone https://github.com/next-theme/hexo-theme-next themes/next
```
